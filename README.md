## Featured Projects

### 🎮 [RealG Optimizer](https://github.com/geraldojoao/windows-gaming-optimizer)

> Windows gaming performance tool focused on lower input latency, frametime stability and safe, reversible system tuning.

- Detects CPU, GPU vendor, RAM, disk type and Windows version automatically
- Applies 12 optimization categories: power, GPU, network, memory, disk, Windows gaming features
- Every change is backed up before writing — full rollback with `[R]` in the menu
- Profiles: Gaming, Competitive, Balanced, Laptop
- Synthetic benchmark score (0–100) with before/after delta
- Persistent log · Windows Restore Point creation · CI with PSScriptAnalyzer

### 📱 [FlutterPOKEDEX](https://github.com/geraldojoao/flutterPOKEDEX)

> Full-featured Pokédex app consuming the public PokeAPI, with type filtering, evolution chains, animated stat bars and favorites management.

- Search, filter by type and sort by name — responsive grid layout
- Detail screen with evolution chain, animated progress bars per stat and Hero transitions
- Favorites screen with persistent state
- Daily random Pokémon card on the home screen
- Light/dark theme with Provider + SharedPreferences
